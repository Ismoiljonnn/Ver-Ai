# 🚀 VerAi: The Global Truth Guardian

**VerAi** — bu internetdagi ma'lumotlar immuniteti. AI texnologiyalari orqali feyk xabarlarni aniqlash va haqiqatni himoya qilish uchun yaratilgan ochiq manbali Ai agent.



## 🔥 Nega VerAi?
* **Zero-Knowledge needed:** Foydalanuvchilar uchun hech qanday API kalit shart emas.
* **Minimalist Design:**Dark and Day Mode.
* **Powered by Groq:** LPU (Language Processing Unit) texnologiyasi bilan bir zumda tahlil.

---

## 🛠 O'rnatish (2 qadam)

### 1. Repozitoriyani nusxalash (Git Clone)
Loyihani kompyuteringizga "clone" qiling va ichiga kiring:
```bash
git clone https://github.com/ismoiljon/verai-detector.git
cd verai-detector
```

### 2. API Kalitni sozlash
`app.py` faylini oching va `GROQ_API_KEY` o'zgaruvchisiga o'z kalitingizni kiriting:
```python
GROQ_API_KEY = 'gsk_your_api_key_here'
```
*Kalitni bepul olish: [console.groq.com](https://console.groq.com)*

---

## 🚀 Ishga tushirish

Kutubxonalarni yuklang va serverni yoqing: 
```bash
pip install -r requirements.txt
python app.py
```
Keyin brauzerda oching: `http://localhost:3000` ✅
