# Fake News Detector 🔍 (Python version)

AI-powered fake news detector in Uzbek. No API key needed for users!

## Setup (2 steps only)

### 1. Add your Groq API key
Open `app.py` and replace `YOUR_GROQ_API_KEY_HERE` with your key:
```
GROQ_API_KEY = 'gsk_your_key_here'
```
Get a free key at: https://console.groq.com

### 2. Run the app
```bash
pip install -r requirements.txt
python app.py
```

Then open http://localhost:3000 in your browser. That's it! ✅
